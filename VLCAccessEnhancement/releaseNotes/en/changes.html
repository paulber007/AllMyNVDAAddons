1. Version 2.13 (June/05/2025)

Compatibility with NVDA 2025.1

NVDA versions prior to 2024.1 are no longer supported.

Bug fixes:

  • the update search is no longer done when the "Automatically search for updates" option is unchecked just after starting NVDA.

Development :

  • when installing the add-on, the current configuration is no longer saved in the add-on folder but is now saved in the user configuration folder. This allows you to still keep the current configuration if an installation goes wrong.

Compliance with the NVDA API:

  • Use of the "ExecandPump" method of the "Systemutils" module.
  • Taking into account of guy's obsolescence.
    • Messagedialog is now used for all NVDA versions greater than 2025.1.

2. Version 2.12 (October/11/2024)

Compatibility with NVDA 2024.2, 2024.3 and 2024.4. Bug fixes:

  • in edit boxes, "space" characters and uppercase characters are now correctly announced.
  • correction of a bad French translation.

3. Version 2.11 (March/27/2024)

Compatibility:

  • Compatibility with NVDA 2024.1.
  • NVDA versions prior to 2023.1 are no longer supported.

What's new:

  • according to the addon store since NVDA 2023.2, it is now possible to update the extension with an incompatible version.
  • taking into account the "Speech on demand" mode introduced in nvda 2024.1: all scripts announcing information will vocalize this information. In this mode, all other scripts will remain silent.

Bug fixes:

  • improved initialization of the extension when playback of a video file is started from Windows Explorer and the cursor remains on the Explorer. But doesn't work in all cases.

4. Version 2.10 (February/18/2023)

Compatibility with NVDA 2023.1

Bug fixes:

  • correction of not returning to the configuration profile of the previous application when stopping VLC or switching with "alt+tab".
  • correction of the impossibility of using VLC with a language not provided for in the extension.

5. Version 2.9 (May/30/2022)

  • the list of recent files with their elapsed times is now correctly found.

  • fixed inability to view extension history with "View History" button.

6. Version 2.8 (February/04/2022)

25. Version 1.0 (10/29/2018)

To avoid confusion with other extensions, for VLC, the name of the extension is to be renamed "VLCAccessEnhancement" and in the extensions manager, it is called "VLC media Player: add-ins of accessibility".

What's new:

  • compatibility with NVDA 2018.3,
  • change extension name to avoid confusion with other extensions for VLC.
  • announcement of the indication of the possibility of the resumption of the playback interrupted position stored by VLC and resuming playback using the gesture command "Alt + Control + r",
  • added button to delete the configuration file of VLC,

Internal change:

  • complete overhaul of the code,
  • file style.css renamed style_md.css,
  • conversion of the file user manual for compliance of the form with the extensions international,
  • renaming of the menu configuration of the extension.

26. Previous history

26.1. Version 3.0 (19/06/2018)

This version is compatible with VLC 3.0, which is incompatible with the old versions.

New:

  • vocalization of the indicator of repetition of the media,
  • correct reading of the status bar,
  • announcement of the state play or pause with the sound muted when the focus of the main window.

Changes:

  • the configuration file of VLC is no longer changed automatically to set the keyboard shortcuts. Their implementation is done manually by the user using a simple button,
  • the dialog box "Go to time" of VLC is no longer vocalized.
  • the volume level is now announced at each change.

correction of bugs (regression of "NVDA + Control + h")
