1. Version 2.13 (05/06/2025)

Compatibilidad con NVDA 2025.1

Las versiones de NVDA anteriores a 2024.1 ya no son soportadas.

Corrección de fallos:

  • la búsqueda de actualización ya no se realiza cuando la opción "Buscar automáticamente actualizaciones" esta sin marcar justo después de iniciar NVDA.

Desarrolladores:

  • al instalar el complemento, la configuración actual ya no se guarda en la carpeta del complemento pero ahora se guarda en la carpeta de configuración del usuario. Esto permite si una instalación sale mal de conservar de todos modos la configuración actual.

Puesta en conformidad con la API de NVDA:

  • Uso del método "ExecAndPump" del módulo "systemUtils".
  • se toma en cuenta la obsolescencia de gui.messageBox y gui.message.messageBox, reemplazado por MessageDialog:
    • MessageDialog ahora se usa para todas las versiones de NVDA superiores a 2025.1.

2. Versión 2.12 (11/10/2024)

Compatibilidad con NVDA 2024.2, 2024.3 y 2024.4. Corrección de fallos:

  • corrección de una mala traducción al francés.

3. Versión 2.11 (27/03/2024)

Compatibilidad:

  • compatibilidad con NVDA 2024.1.
  • las versiones de NVDA anteriores a la 2023.1 ya no son soportadas.

Nuevas características:

  • de acuerdo con la Tienda de complementos (Add-on Store) desde NVDA 2023.2, ahora es posible actualizar el complemento con una versión incompatible.
  • se toma en cuenta un nuevo modo de "voz a petición" introducido en nvda 2024.1: todos los scripts que anuncien información verbalizarán esta información. En este modo, todos los demás scripts permanecerán en silencio.
  • se toma en cuenta un nuevo modo de "voz a petición" traído por nvda 2024.1.

Corrección de fallos:

  • mejora de la inicialización del complemento cuando se inicia la reproducción de un archivo de vídeo desde el Explorador de Windows y el cursor permanece en el Explorador. Pero no funciona en todos los casos.

25. Versión 1.0 (29/10/2018)

Para evitar confusiones con otros complementos para VLC, el nombre del complemento cambia a "VLCAccessEnhancement" y en el Administrador de Complementos, se llama "Reproductor multimedia VLC: complementos de accesibilidad".

Novedades:

  • Puesta en compatibilidad con NVDA 2018.3,
  • Cambio en el nombre del complemento para evitar confusiones con otros complementos para VLC.
  • Anuncia la indicación de la posibilidad de reanudar la reproducción interrumpida en la posición memorizada por VLC y reanuda la reproducción con el gesto de entrada "alt+control+r",
  • Se añade el botón para eliminar el archivo de configuración de VLC,

Cambio interno::

  • Reelaboración completa del código,
  • Archivo style.css renombrado en style_md.css,
  • Reconversión del archivo del manual del usuario para la conformidad en la forma con los complementos internacionales,
  • Renombrado el menú de configuración del complemento.

26. Historial anterior

26.1. Versión 3.0 (19/06/2018)

Esta versión es compatible con VLC 3.0, incompatible con versiones anteriores.

Novedades:

  • Verbalización del indicador de repetición del medio,
  • Lectura correcta de la barra de estado,
  • Anuncio del estado reproduciendo o en pausa con el sonido silenciado al enfocar la ventana principal.

Cambios:

  • El archivo de configuración de VLC ya no se modifica automáticamente para definir atajos de teclado. Su implementación se realiza manualmente por el usuario con la ayuda de un simple botón,
  • El cuadro de diálogo "Ir a tiempo" de VLC ya no se verbaliza.
  • El nivel de volumen ahora se anuncia con cada cambio.
