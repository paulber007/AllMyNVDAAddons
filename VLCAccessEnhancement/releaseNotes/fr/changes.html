1. Version 2.13 (05/06/2025)

Compatibilité avec NVDA 2025.1

Les versions de NVDA antérieures à la 2024.1 ne sont plus supportées.

Correction de bogues :

  • la recherche de mise à jour ne se fait plus lorsque l'option "Rechercher automatiquement les mises à jour " est décochée juste après le démarrage de NVDA.

Développement :

  • lors de l'installation de l'extension, la configuration courante n'est plus sauvegardée dans le dossier de l'extension mais est maintenant sauvegardée dans le dossier de la configuration utilisateur. Cela permet si une installation se passe mal de conserver quand même la configuration courante.

Mise en conformité avec l'API de NVDA :

  • utilisation de la méthode "ExecAndPump" du module "systemUtils ".
  • prise en compte de l'obsolescence de gui.messageBox et gui.message.messageBox, remplacés par MessageDialog:
    • MessageDialog est maintenant utilisé pour toutes les versions de NVDA supérieures à 2025.1.

2. Version 2.12 (11/10/2024)

Compatibilité avec NVDA 2024.2, 2024.3 et 2024.4. Corrections de bogues :

  • dans les zones d'édition, les caractères "espace" et caractères en majuscule sont maintenant correctement annoncées.
  • correction d'une mauvaise traduction française.

3. Version 2.11 (27/03/2024)

Compatibilité :

  • compatibilité avec NVDA 2024.1.
  • les versions de NVDA antérieures à la 2023.1 ne sont plus soutenues.

Nouveautés :

  • conformément à l'addon store depuis NVDA 2023.2, il est maintenant possible de mettre à jour l'extension avec une version incompatible.
  • prise en compte du mode "Parole à la demande" introduit dans nvda 2024.1: tous les scripts annonçant une information vocaliseront cette information. Dans ce mode, tous Les autres scripts resteront muets.
  • prise en compte du nouveau mode "Parler à la demande" amené par nvda 2024.1.

Correction de bogues :

  • amélioration de l'initialisation de l'extension quand la lecture d'un fichier vidéo est lancée depuis l'explorateur Windows et que le curseur reste sur l'explorateur. Mais ne marche pas dans tous les cas.

4. Version 2.10 (18/02/2023)

Compatibilité avec NVDA 2023.1

Correction de bogues:

  • correction du non retour au profil de configuration de l'application précédente lors de l'arrêt de VLC ou par commutation par "alt+tab".
  • correction de l'impossibilité d'utiliser VLC avec un langage non prévu dans l'extension.

5. Version 2.9 (30/05/2022)

  • la liste des fichiers récents avec leurs temps écoulés est maintenant correctement retrouvée.
  • annonce du volume lors d'une demande d'augmentation ou diminution, lorsque aucun média est en cours.
  • les messages vocaux sont maintenant aussi envoyés à l'afficheur braille.
  • correction de l'impossibilité d'afficher l'historique de l'extension avec le bouton "Voir l'historique".

6. Version 2.8 (04/02/2022)

  • compatibilité avec NVDA 2022.1.
  • réinitialisation de la configuration en cas d'erreurs dans le fichier de configuration.

Changements:

  • lors d'une mise à jour automatique de l'extension, il n'est plus demander à l'utilisateur s'il veut conserver les paramètres actuels de configuration de l'extension.
  • lors d'une installation manuelle de l'extension, une demande de confirmation est demandée qquand l'utilisateur a choisi de ne pas garder les paramètres actuels de la configuration de l'extension.

25. Version 1.0 (29/10/2018)

Pour éviter une confusion avec d'autres extensions pour VLC, le nom de l'extension est renommé en "VLCAccessEnhancement" et dans le gestionnaire d'extensions, il se nomme "Lecteur multimédia VLC: compléments d'accessibilité".

Nouveautés:

  • mise en compatibilité avec NVDA 2018.3,
  • changement du nom de l'extension pour éviter toute confusion avec d'autres extensions pour VLC.
  • annonce de l'indication de la possibilité de reprise de la lecture interrompue à la position mémorisée par VLC et reprise de la lecture à l'aide du geste de commande "Alt + Control + r",
  • ajout du bouton pour supprimer le fichier de configuration de VLC,

Changement interne:

  • remaniement complet du code,
  • fichier style.css renommé en style_md.css,
  • reconversion du fichier manuel utilisateur pour conformité de forme avec les extensions internationales,
  • renommage du menu de configuration de l'extension.

26. Historique précédent

26.1. Version 3.0 (19/06/2018)

Cette version est compatible avec VLC 3.0, incompatible avec les anciennes versions.

Nouveauté:

  • vocalisation de l'indicateur de répétition du média,
  • lecture correcte de la barre d'état,
  • annonce de l'état lecture ou pause avec le son coupé lors de la focalisation de la fenêtre principale.

Changements:

  • le fichier de configuration de VLC n'est plus modifié automatiquement pour définir les raccourcis clavier. Leur mise en place est faite manuellement par l'utilisateur à l’aide d'un simple bouton,
  • la boite de dialogue "Aller au temps" de VLC n'est plus vocalisée.
  • le niveau du volume est maintenant annoncé à chaque modification.

correction de bugs (régression de "NVDA + Control + h")
