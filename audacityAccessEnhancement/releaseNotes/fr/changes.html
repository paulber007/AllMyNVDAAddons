1. v3.3 (05/06/2025)

Compatibilité avec NVDA 2025.1.

Les versions de NVDA antérieures à la 2024.1 ne sont plus supportées.

Compatibilité avec audacity 3.7.1. les version inférieures à la version 3.5.1 et les versions 32 bits ne sont plus supportées.

Correction de bogues :

  • la recherche de mise à jour ne se fait plus lorsque l'option "Rechercher automatiquement les mises à jour " est décochée juste après le démarrage de NVDA.

Développement :

  • lors de l'installation de l'extension, la configuration courante n'est plus sauvegardée dans le dossier de l'extension mais est maintenant sauvegardée dans le dossier de la configuration utilisateur. Cela permet si une installation se passe mal de conserver quand même la configuration courante.

Mise en conformité avec l'API de NVDA :

  • utilisation de la méthode "ExecAndPump" du module "systemUtils ".
  • prise en compte de l'obsolescence de gui.messageBox et gui.message.messageBox, remplacés par MessageDialog:
    • MessageDialog est maintenant utilisé pour toutes les versions de NVDA supérieures à 2025.1.

2. v3.2 (11/10/2024)

  • compatibilité avec NVDA 2024.2, 2024.3 et 2024.4.
  • compatibilité et tests avec Audacity 3.6.4 x64, 3.6.2 x64 et 3.5.1 x64.
  • mise à niveau du guide d'introduction à Audacity.

3. v3.1 (23/03/2024)

Compatibilité avec NVDA 2024.1

Nouveautés :

  • prise en compte du mode "Parole à la demande" introduit dans nvda 2024.1: tous les scripts annonçant une information vocaliseront cette information. Dans ce mode, tous Les autres scripts resteront muets.

Correction de bogues :

  • le bouton radio permettant de choisir le type d'exportation (exporter le projet entier, export de fichiers multiple, ...) est maintenant accessible.

4. v3.0 (23/11/2023)

Nouveautés:

  • nomage des panneaux de barres d'outils en "paneaux inférieur" et "Panneau supérieur".
  • nommage du panneau contenant la vue de piste en "Panneau principal".

Changements:

  • Les scripts pour annoncer le début, la fin la durée et le centre de la sélection sont remplacés par :
    • un script pour annoncer la sélection,
    • un script pour annoncer le premier temps de la sélection,
    • un script pour annoncer le second temps de la sélection.
  • dans la documentation, la terminologie "zone de sélection numérique" est remplacée par "boite de sélection numérique" conformément au guide d'Audacity.
  • dans la documentation, le terme groupe" pour les barre d'outils est remplacé par "panneau".
  • la fonctionnalité "Vocalisation des boites de sélection numérique du temps Audacity " n'est pas disponible dans les versions 3.3.x d'Audacity.

Compatibilité:

  • Cette version de l'extension a été testée avec Audacity 3.4, 3.4.2, 3.3.0 et 3.3.3. Elle devrait aussi être compatible avec les versions 3.3.1 et 3.3.2, mais aucun test n'a été fait.
  • La compatibilité avec les versions antérieures à la 3.3.0 n'est plus assurée.
  • les versions de NVDA inférieures à la 2022.1 ne sont plus supportés.

5. v2.9 (03/08/2023)

  • la position audio n'est plus annoncé après la relance de la lecture. Cela se produisait avec les versions 3.2.2 et supérieures d'Audacity. Dans ces versions , l'état des boutons "lecture" et "Pause" n'est plus disponible avec MSAA), contrairement aux précédentes versions.
