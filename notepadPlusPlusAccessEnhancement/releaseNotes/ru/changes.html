Текстовый редактор Notepad++ : улучшение доступности - История изменений

Автор: paulber19 (paulber19@laposte.net)

URL: https://github.com/paulber19/notepadPlusPlusAccessEnhancementNVDAAddon.git

### Загрузка:

  • Текущая стабильная версия: 2.8.4
  • Предыдущая стабильная версия: 2.7.2
  • Разрабатываемые версии:

    ### v2.8 (05/06/2025) + Совместимость с NVDA 2025.1.

    Версии NVDA старше 2024.1 больше не поддерживаются.

    Нововведения:
  • Добавлен скрипт "windows+control+f9" для открытия папки текущего файла в качестве рабочей области.
  • В диалоге "Предпросмотр документа" добавлена возможность выбора способа просмотра:
    • Браузер по умолчанию для HTML-документов,
    • Окно NVDA в режиме навигации,
    • Блокнот по умолчанию для просмотра исходного кода сгенерированного документа.

      Исправления:
  • Проверка обновлений больше не выполняется, если опция "Автоматически проверять обновления" отключена сразу после запуска NVDA.
  • Номер строки теперь корректно озвучивается для файлов, использующих только "\r" как конец строки.

    Разработка:
  • Обработка текущего документа с помощью компилятора markdown и всех его расширений вместо markdown2.
  • Использование "NVDAState.shouldWriteToDisk" для управления сохранением конфигурации на диск.
  • Временные файлы для предпросмотра документов "markdown" или "txt2tags" теперь создаются в той же папке, где находится документ. Для несохранённых документов используется папка конфигурации пользователя.
  • При установке дополнения конфигурация теперь сохраняется в папке пользователя, а не в папке дополнения.

    Соответствие API NVDA:
  • Использование метода "ExecAndPump" из модуля "systemUtils".
  • Замена устаревших функций gui.messageBox и gui.message.messageBox на MessageDialog для NVDA 2025.1 и новее.

    ### v2.7 (18/10/2024) + Совместимость с NVDA 2024.2, NVDA 2024.3 и NVDA 2024.4. Версии NVDA старше 2023.1 больше не поддерживаются.

    ### v2.6 (02/04/2024) + Нововведения:
  • Совместимость с NVDA 2024.1. Поддержка версий NVDA старше 2023.1 прекращена.
  • Поддержка режима "Речь по запросу" (введён в NVDA 2024.1): скрипты теперь озвучивают информацию только в этом режиме.

    ### v2.5 (08/09/2023) + Нововведения:
  • Возможность озвучивания пути к файлу с учётом иерархии папок.
  • Улучшение доступности диалога "Документы" (открывается через "Alt+d" → "d"):
    • Сокращённое озвучивание пути,
    • Устранение дублирования озвучивания выбранного элемента,
    • Возможность выбора файла по первой букве имени,
    • Настройка озвучивания пути, типа и размера файла.

      Совместимость:
  • Поддержка NVDA ниже 2022.1 прекращена.
  • Совместимость с Notepad++ 8.5.6.

    Исправления:
  • Корректное перемещение к первому символу после максимальной длины строки для строк с акцентами.
  • Исправлено уведомление о длинных строках при перемещении курсора влево/вправо.
  • Устранён двойной звуковой сигнал при переходе к символу за пределом длины строки.

    Прочее:
  • Обновление markdown2.py до версии 2.4.9.
  • Удаление кода для Python 2.
  • Адаптация buildVars.py для Add-on Store.
  • Обновление документации.

    ### v2.4 (20/02/2023) + Совместимость с NVDA 2023.1. Нововведения:
  • Уведомление об орфографических ошибках, найденных модулем DSpellCheck (экспериментальная функция).

    ### v2.3 (27/09/2022) + Исправления:
  • Устранена ошибка при открытии истории изменений через кнопку "Посмотреть историю".
  • Исправлено некорректное перемещение между методами и классами.
  • Идентификация строк методов и классов теперь происходит только по "def " и "class " соответственно.
  • Совместимость с Notepad++ 8.3 (64 бита).

    ### v2.2 (04/02/2022) + Совместимость с NVDA 2022.1.
  • Сброс конфигурации при ошибках в файле настроек.

    Изменения:
  • При автоматическом обновлении больше не запрашивается подтверждение сохранения настроек.
  • При ручной установке добавлено подтверждение при отказе от сохранения текущих настроек.

    ### v2.1 (13/12/2021) + Первая официальная стабильная версия.
