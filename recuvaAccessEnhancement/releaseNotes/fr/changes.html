Auteur : paulber19 (paulber19@laposte.net)

Téléchargement :

  • Version stable actuelle : 1.9
  • Version stable précédente : 1.8
  • versions de développement

1. v1.9 (18/10/2024)

Compatibilité avec NVDA 2024.2, NVDA 2024.3 et NVDA 2024.4.

Les versions de NVDA antérieures à la version 2023.1 ne sont plus prises en charge.

2. v1.8 (25/03/2024)

Compatibilité avec NVDA 2024.1

Nouveautés :

  • prise en compte du mode "Parole à la demande" introduit dans nvda 2024.1: tous les scripts annonçant une information vocaliseront cette information. Dans ce mode, tous Les autres scripts resteront muets.
  • compatibilité de l'extension avec la version 64 bits de Recuva.

Développement :

  • utilisation du décorateur scriptHandler.script.
  • suppression du code spécifique aux versions antérieures à NVDA 2022.1.

3. v1.7 (14/02/2023)

Compatibilité avec NVDA 2023.1

4. v1.6 (31/05/2022)

  • correction de l'impossibilité d'afficher l'historique de l'extension à l'aide du bouton "Voir l'historique".

5. v1.5 (27/01/2022)

  • Compatibilité avec NVDA 2022.1,
  • réinitialisation de la configuration en cas d'erreurs dans le fichier de configuration.

Changements:

  • lors d'une mise à jour automatique de l'extension, il n'est plus demander à l'utilisateur s'il veut conserver les paramètres actuels de configuration de l'extension.
  • lors d'une installation manuelle de l'extension, une demande de confirmation est demandée qquand l'utilisateur a choisi de ne pas garder les paramètres actuels de la configuration de l'extension.

6. v1.4 (04/11/2021)

compatibilité avec NVDA 2021.3. Les versions de NVDA antérieures à la version 2020.4 ne sont pas supportées.

7. v1.3 (22/06/2021)

  • compatibilité avec NVDA 2021.1. Les versions de NVDA antérieures à la version 2019.3 ne sont pas supportées.
  • suppression du code spécifique à Python 2.7.

8. v1.2.1 (04/02/2021)

Ajout traduction arabe

9. v1.2 (29/12/2020)

  • compatibilité avec NVDA 2020.4
  • ajout du bouton "Voir l'historique" dans la catégorie "Mise à jour " des paramètres de l'extension.

10. v1.1.1 (29/10/2020)

Correction de la documentation

11. v1.1 (27/10/2020)

  • compatibilité avec NVDA 2020.3,
  • nettoyage du code (corrections des erreurs détectées par Flake8)
  • ajout du bouton "Quoi de neuf" dans le dialogue proposant l'installation d'une nouvelle mise à jour.

12. v1.0 (28/07/2020)

Création de l'extension.
