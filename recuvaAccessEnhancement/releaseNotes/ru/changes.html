Автор: paulber19 (paulber19@laposte.net)

URL: https://github.com/paulber19/recuvaAccessEnhancementNVDAAddon.git

### Загрузка:

  • Текущая стабильная версия: 1.10
  • Предыдущая стабильная версия: 1.9.1
  • Разрабатываемые версии

    ### + v1.10 (05/06/2025) + Совместимость с NVDA 2025.1

    Версии NVDA старше 2024.1 больше не поддерживаются.

    Исправления ошибок:
  • Проверка обновлений больше не выполняется, если опция "Автоматически проверять обновления" отключена сразу после запуска NVDA.

    Разработка:
  • При установке дополнения текущая конфигурация больше не сохраняется в папке дополнения, а перенесена в папку пользовательской конфигурации. Это позволяет сохранить настройки даже в случае неудачной установки.

    Соответствие API NVDA:
  • Использование метода "ExecAndPump" из модуля "systemUtils".
  • Учёт устаревания gui.messageBox и gui.message.messageBox, заменённых на MessageDialog:
    • MessageDialog теперь используется для всех версий NVDA выше 2025.1.

      ### + v1.8 (25/03/2024) + Совместимость с NVDA 2024.1

      Нововведения:
  • Поддержка режима "Речь по запросу" (введён в NVDA 2024.1): все скрипты, объявляющие информацию, будут её озвучивать. В этом режиме остальные скрипты останутся беззвучными.
  • Совместимость с 64-битной версией Recuva.

    Разработка:
  • Использование декоратора scriptHandler.script.
  • Удаление кода для версий NVDA старше 2022.1.
