Автор: paulber19 (paulber19@laposte.net)

URL: https://github.com/paulber19/wordAccessEnhancementNVDAAddon.git

### v3.7 (06/06/2025) Совместимость:

  • Добавлена поддержка NVDA 2025.1
  • Версии NVDA ниже 2024.1 больше не поддерживаются

    Нововведения:
  • Скрипты для перемещения между орфографическими/грамматическими ошибками без режима навигации
  • Скрипт отображения исправлений для слов с ошибками
  • Озвучивание действий комбинаций Ctrl+Shift+O/P (режимы Word)
  • Указание типа представления в описании документа
  • Скрипт переключения между голосами

    Изменения:
  • Команды перемещения в таблицах не блокируют системные сочетания Win+Ctrl+Стрелки
  • Удален параметр словарей символов из настроек голоса
  • Сброс голосовых настроек для NVDA 2025.1

    Исправления:
  • Корректная работа автопроверки обновлений
